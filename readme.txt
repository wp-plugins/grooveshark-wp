== Description ==

This Grooveshark plugin allows you to show your recently played Grooveshark songs and display them in a list (With links) as a widget on your blog.

You just need to insert your Grooveshark username and the number of links you want in the list, and everything will be managed for you. You just need to enable the widget.

Installing it is also very simple. See the installation menu for instructions.

== Installation ==

1. Upload `grooveshark-wp.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the Grooveshark WP Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets tab.
